# Introduction à Git & GitHub

## 1. Description
Ce projet présente les bases de Git et GitHub.

## 2. Prérequis
- Git
- VS Code
- Un compte GitHub

## 3. Utilisation

Ajouter les fichiers :
```bash
git add .
```

Créer un commit :
```bash
git commit -m "Premier commit"
```

Envoyer vers GitHub :
```bash
git push origin main
```

Récupérer les modifications :
```bash
git pull
```
